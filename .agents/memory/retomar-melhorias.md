# Comando: "reiniciar as melhorias"

## Regra
Quando o usuário disser a frase exata **"reiniciar as melhorias"**, retomar a implementação de novas features do NEXUS ERP a partir do ponto onde foi pausado.

## Por que
O usuário pediu para parar de implementar e entrar em modo de testes. Quando quiser retomar, usará essa frase como gatilho.

## Como aplicar
1. Verificar as tasks pendentes/propostas no sistema de project tasks
2. Retomar a partir da primeira task não concluída ou proposta que ainda não foi iniciada
3. Continuar o ciclo normal de desenvolvimento (implementar → merge → próxima task)

**Why:** Preferência explícita do usuário registrada em 06/06/2026.
